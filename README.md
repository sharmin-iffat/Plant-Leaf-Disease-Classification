# 🌿 Plant Leaf Disease Classification (Deep Learning Project)

A deep learning-based web application that classifies plant leaf diseases using EfficientNet-B0 and transfer learning. The model is trained on labeled leaf images to accurately detect and categorize various plant diseases, achieving ~90% accuracy.

## 🚀 Features
- CNN-based classification using EfficientNet-B0  
- Transfer learning for improved performance on limited data  
- Image preprocessing (resizing, normalization)  
- Data augmentation (rotation, flipping) for better generalization  
- Fine-tuned with Adam optimizer and CrossEntropy loss  
- Real-time prediction using OpenCV  

## 🛠️ Tech Stack
- Python  
- PyTorch  
- OpenCV  
- Deep Learning  

## 📊 Model Performance
- Accuracy: ~90%  
- Robust against variations in lighting and orientation  

## 📌 Use Cases
- Early detection of plant diseases  
- Smart agriculture solutions  
- Assistance for farmers and researchers
